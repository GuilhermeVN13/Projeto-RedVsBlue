# 🔴🔵 Red vs Blue Team Cybersecurity Lab

This repository contains practical cybersecurity labs focused on **offensive (Red Team)** and **defensive (Blue Team)** techniques.

The objective of this project is to simulate real-world attack scenarios and demonstrate how malicious activity can be **detected and analyzed using SIEM and log monitoring tools**.

---

# 🎯 Project Goals

* Simulate real-world cyber attacks
* Practice offensive security techniques using Kali Linux
* Detect malicious activity using SIEM
* Analyze security logs from compromised systems
* Map attacker behavior using the MITRE ATT&CK framework
* Build hands-on experience for SOC Analyst and Blue Team roles

---

# 🧪 Lab Architecture

The lab environment uses **Kali Linux as the main host system**, responsible for launching attacks and analyzing logs.

Virtual machines are used as **target systems and monitored endpoints**.

| System            | Role                         |
| ----------------- | ---------------------------- |
| Kali Linux (Host) | Attacker + Wazuh Manager     |
| Windows VM        | Target machine + Wazuh Agent |
| Linux VM          | Target machine + Wazuh Agent |

All machines communicate within an **isolated virtual network** to safely simulate attacks.
